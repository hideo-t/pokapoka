## 📅 ごよやくのしかた（くわしく）

### ステップ1: ごれんらくさきのメールアドレスをにゅうりょく
- ふだんつかっている **メールアドレス** をにゅうりょく
- 「コードをおくる」をタップ

### ステップ2: メールにとどく「にんしょうコード」をにゅうりょく
- 1〜2ふんでメールがとどきます
- メールにかかれた **6けたのすうじ** をがめんににゅうりょく
- 「にんしょうする」をタップ

**コードのゆうこうきげんは 10ぷん** です。  
10ぷんいじょうたつとつかえなくなるので、もういちど「コードをおくる」をしてください。

#### メールがとどかないとき
- 「めいわくメールフォルダ」をみてください
- それでもとどかないときは、お問い合わせ画面から ご連絡ください

### ステップ3: かいさいびをえらぶ
- 「うけつけちゅう」のひだけえらべます
- 「のこり: おとな N めい / こども M めい」で あきじょうきょうがわかります

| バッジ | いみ |
|---|---|
| 🟢 うけつけちゅう | よやくOKです |
| ✅ よやくずみ | あなたはすでにごよやくずみ |
| 🟠 まんせき・キャンセルまち | まんせきですが、キャンセルまちで とうろくできます |
| ⛔ うけつけしてい | このひはおやすみです |

### ステップ4: にんずう・おなまえ・アレルギーをにゅうりょく
- **おなまえ** はひつようです
- **でんわばんごう** はにんいです（ごれんらくがひつようなとき につかいます）
- **おとな・こども のにんずう** をしてい
- **アレルギー・きをつけたいこと** をかいてください（れい: たまご・小麦・乳製品）
- **どこでしりましたか？** （しょかいのかただけ）

### ステップ5: かくにんがめん
- ないようをみて、ただしければ「よやくする」をタップ
- まちがいがあれば「←しゅうせいする」でまえのがめんへ

### ステップ6: かんりょう！
- 🎉 「よやくがかんりょうしました！」のがめんがでます
- **よやくばんごう（8けたの えいすうじ）** がひょうじされます
- どうじに **かくにんメール** がとどきます — メールはたいせつにほぞんしてください

## 📋 マイよやく（じぶんのよやくをみる）

### マイよやくをひらく
1. メールにんしょうしてログインします
2. みぎうえの **☰ メニュー** をタップ
3. **📋 マイよやく** をタップ

### マイよやくでできること
- **これからのよやく** と **これまでのりよう** にわけてひょうじ
- それぞれに **キャンセルボタン**
- 「らいじょう N かいめ」「ぜんかい: YYYY/MM/DD」などをひょうじ

### ごよやくをキャンセル
1. マイよやくをひらく
2. キャンセルしたいごよやくの「キャンセル」ボタンをタップ
3. かくにんがでるので「OK」をタップ
4. キャンセルメールがとどきます

### じょうれんさんバッジ 🌟
**らいじょう 10かいめ** をこえると、「🌟 じょうれんさん」とひょうじされます。  
いつもありがとうございます。

3. うけつけかんりょう。⏳ マークでひょうじされます
4. キャンセルがでて、まんがいち くりあげになったら **メールでおしらせ** します

### Q. よやくしたメールがみつかりません
**A.** 「めいわくメールフォルダ」をご確認ください。  
それでもみつからないときは、おといあわせから ごれんらくください。**よやくばんごう** をおぼえていれば、すぐにかくにんできます。

### Q. ログインしなくても かいさいびはみれますか？
